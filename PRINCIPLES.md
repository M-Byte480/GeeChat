# 1 Data is in your control
- We don't have a centralized database of user data. We don't have access to your data
- Only the server owner has access to your IP address and the data you send to the server.
# 2 We avoid enshitifying your experience
- We prioritize your experience and control over the server experience. We don't want to add features that make the server experience worse for you, even if it makes it better for the server owner.
# 3 You control what you see
- Because we don't centralize hosting the server, and to give you control it ends up you will have to self host what you want to be able to use and see
